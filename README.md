# Dropbox Clone

Dropbox clone using NodeJS, MongoDB, React, and React Native

----

**Create Box**
----

* **URL**

    `BASE_URL/boxes`

* **Method:**

    `POST`


**Upload File**
----

* **URL**

    `BASE_URL/boxes/:id/files`

* **Method:**

    `POST`

*  **URL Params**

   **Required:**
 
   `id=[hash]`


**List Files in Box**
----

* **URL**

    `BASE_URL/boxes/:id`

* **Method:**

    `GET`

*  **URL Params**

   **Required:**
 
   `id=[hash]`
